Task Tracer 是一个面向个人使用的任务工作台。它可以作为可安装的网页应用运行，任务数据保存在当前浏览器中，适合日常任务捕获、项目跟进、重复事项、本地提醒、任务归档和个人效率回顾。

它适合学习计划、个人事务、独立项目、例行复盘、截止日期工作，以及任何更重视本地掌控、清晰状态和数据可导出的任务管理场景。

## 可以做什么

| 需求 | Task Tracer 支持 |
| --- | --- |
| 快速记录任务 | 完整任务表单、快速添加解析、截止日期/时间、无日期任务、子任务、项目和标签。 |
| 管理不断增长的列表 | 项目分组、多标签标记、限定筛选、智能搜索、归档历史和手动排序。 |
| 按时间规划 | 列表、日历、时间线、实时倒计时、状态片段、进度条和跨时区稳定日期。 |
| 处理例行事项 | 每天、每周、指定星期、每月、每月最后一天和自定义周期重复。 |
| 获得提醒 | 浏览器提醒、提前提醒、重复提醒、稍后提醒、错过提醒提示和后台提醒限制说明。 |
| 回顾完成情况 | 完成率、进行中任务逾期率、归档数量、今日完成数、连续完成天数和近期趋势。 |
| 保护数据 | 导出、导入预览、合并/替换冲突处理、带版本说明的备份、备份健康状态和存储受限时的紧急备份。 |
| 舒服地使用 | 响应式布局、明暗主题、减少动态效果、键盘导航、读屏标签和中英文界面。 |

## 日常工作流

1. 使用完整弹窗添加任务，或用快速添加输入一条轻量任务。
2. 在列表视图中完成、编辑、稍后提醒、归档、删除、处理子任务和手动排序。
3. 当日期排布比列表顺序更重要时，切换到日历视图或时间线视图。
4. 在统计视图中查看当前负载、完成表现、归档历史和近期状态。
5. 在更换浏览器、清理站点数据或测试导入文件前，先导出或备份。

## 快速添加

快速添加可以把轻量文本解析成结构化任务字段。

```text
明天 20:00 复习英语 #学习 /个人
```

这会创建一个名为“复习英语”的任务，截止到明天 20:00，标签为“学习”，项目为“个人”。

命令面板使用 `Ctrl/Cmd + P` 打开，可以新建任务、聚焦快速添加、切换视图、导出、备份、搜索和跳转项目。

## 项目、标签与搜索

项目是主要分组。每个任务属于一个项目，例如 `工作`、`个人` 或某个具体计划。项目选择器可以收窄任务空间，全部项目列表会按项目保持分组。

标签是灵活的辅助标记。一个任务可以有多个标签，例如 `学习`、`设计`、`复盘`。标签适合连接不同项目中的同类任务。

搜索会匹配任务名称、任务描述、项目名称、标签和子任务文本。限定搜索可以和普通关键词一起使用：

| 输入 | 匹配内容 |
| --- | --- |
| `project:工作` | 项目名包含“工作”的任务。 |
| `tag:学习` 或 `#学习` | 拥有对应标签的任务。 |
| `status:active` | 进行中的未完成任务。 |
| `status:completed` | 已完成任务。 |
| `status:archived` | 已归档任务。 |
| `status:overdue` | 已过期且未完成的任务。 |
| `status:no-deadline` | 无截止日期任务。 |
| `status:repeat` | 重复任务。 |
| `due:today` | 今天截止的任务。 |
| `due:tomorrow` | 明天截止的任务。 |
| `due:week` | 未来七天内截止的任务。 |
| `due:2025-05-20` | 指定本地日期截止的任务。 |
| `due:no-deadline` | 无截止日期任务。 |
| `project:工作 tag:报告` | 同时满足两个限定条件的任务。 |

如果项目名或标签名包含空格，可以搜索其中有区分度的一部分。

## 重复任务与提醒

重复任务支持每天、每周、指定星期、每月、每月最后一天和自定义天数周期。重复任务可以暂停，也可以跳过本次，而不需要删除重复规则。

当浏览器支持通知且用户授予权限时，Task Tracer 可以在应用打开期间检查提醒、按设定间隔重复提醒、将提醒稍后处理，并在应用重新运行时提示错过的提醒。

浏览器提醒不是系统级闹钟服务。提醒是否准时投递会受到浏览器策略、操作系统行为、电池设置、标签页生命周期，以及应用是否处于打开或被唤醒状态影响。

## 导入、导出与备份

| 动作 | 用途 |
| --- | --- |
| 导出 | 下载当前任务 JSON，适合迁移、人工检查或手动保存。 |
| 导入 | 写入前先预览 JSON 文件，展示任务数量、重复 ID、同名重复项、当前数据影响和同名冲突处理选项。 |
| 备份 | 下载带版本信息的快照，包含结构说明、本地日期、提醒时间、重复规则、完成时间和归档状态。 |

导入可以替换当前任务集合，也可以合并到现有数据中。合并模式下，可以保留两份、跳过导入重复项，或用导入项替换本地同名任务。

备份健康状态会显示在应用菜单中，用来提示今天已备份、最近已备份，或建议重新备份。

## 安装、离线与更新

Task Tracer 是 PWA，也就是浏览器支持时可以像应用一样安装和离线打开的网页应用。应用菜单中提供安装入口。如果当前浏览器没有自动安装弹窗，同一个入口会显示桌面 Chrome/Edge、Android Chrome 和 iOS Safari 的手动安装方式。

应用外壳在缓存后可以离线加载。当有新的离线版本准备好时，Task Tracer 会显示更新提示，并提供刷新更新操作。

## 数据与隐私

任务数据保存在当前浏览器的 IndexedDB 中。Task Tracer 不需要账号，默认不会把任务内容上传到服务器。

如果浏览器阻止本地存储，Task Tracer 会进入存储不可用模式，而不是假装更改已经保存。页面仍保留内存任务快照时，请先下载紧急备份，再刷新或关闭页面。调整浏览器隐私设置后，可以使用“重试”重新检测本地存储。

更换浏览器、清理站点数据、重新安装应用或迁移设备前，请先使用“立即备份”或“导出”下载 JSON 文件。

## 运行方式

直接访问在线地址：

```text
https://todo.muquew.com/
```

或本地运行：

```bash
git clone https://github.com/muquew/Task-Tracer.git
cd Task-Tracer
python3 -m http.server 8080
```

然后打开：

```text
http://127.0.0.1:8080/
```
